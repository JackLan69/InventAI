⬡INVENTAI/COUNCIL
three-agent improvement loop · inventor → researcher → critic → refinement
API key handling: Your key is stored only in your browser's session memory and is sent directly to Anthropic's API. It is never logged or transmitted anywhere else. To use this page you need an Anthropic API key from console.anthropic.com. Each council run costs roughly $0.05–$0.30 depending on model.

How to read the output: Each council proceeds in 4 turns. The Inventor proposes one specific improvement. The Researcher fact-checks for prior art and feasibility. The Critic attacks every weak point. The Inventor returns with a refined version that addresses the critique.
